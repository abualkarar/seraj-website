# السراج للتجهيزات الصناعية | Al-Seraj Industrial Equipment 🏢🔥

الموقع الرسمي لشركة **السراج للتجهيزات الصناعية**، الوكيل المعتمد لشركة **Asenware** العالمية في الجمهورية اليمنية. متخصصون في تقديم أحدث حلول السلامة، أنظمة الإنذار المبكر، ومكافحة الحريق.

[معاينة الموقع المباشر 🌐](https://abualkarar.github.io/seraj-website/)

---

## 🚀 المميزات الجديدة | New Features

- **معرض المشاريع (Portfolio Gallery)**: عرض احترافي لأهم القطاعات التي تخدمها الشركة (مستشفيات، مصانع، فنادق).
- **زر الواتساب العائم**: تواصل فوري ومباشر مع خدمة العملاء من أي مكان في الموقع.
- **كتالوج منتجات ذكي**: عرض مواصفات المنتجات مع إمكانية الاستفسار عن كل منتج بشكل منفصل عبر الواتساب.
- **تحسينات SEO متقدمة**: دعم كامل لمحركات البحث ومنصات التواصل الاجتماعي (Open Graph Tags).
- **حركات تفاعلية (AOS)**: تجربة مستخدم حيوية مع حركات ظهور سلسة عند التمرير.
- **الوضع الليلي والنهاري**: تبديل سلس بين الأوضاع البصرية مع حفظ تفضيلات المستخدم.
- **تصميم زجاجي (Glassmorphism)**: استخدام أحدث صيحات التصميم العالمي في الواجهات.
