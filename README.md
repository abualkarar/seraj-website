# السراج للتجهيزات الصناعية | Al-Seraj Industrial Equipment 🏢🔥

الموقع الرسمي لشركة **السراج للتجهيزات الصناعية**، الوكيل المعتمد لشركة **Asenware** العالمية في الجمهورية اليمنية. متخصصون في تقديم أحدث حلول السلامة، أنظمة الإنذار المبكر، ومكافحة الحريق.

[معاينة الموقع المباشر 🌐](https://abualkarar.github.io/seraj-website/)

---

## 🚀 آخر التحديثات النوعية | Latest Premium Updates

- **شاشة تحميل احترافية (Custom Preloader)**: تجربة دخول فخمة تحمل شعار الشركة.
- **شريط الإعلانات (Announcement Bar)**: للتواصل الفوري بالعروض والخصومات الجديدة.
- **قسم الأسئلة الشائعة (Interactive FAQ)**: إجابات ذكية وتفاعلية لزيادة ثقة العملاء.
- **قسم "من نحن" المطور**: عرض بصري لرؤية وقيم الشركة.
- **زر العودة للأعلى (Back to Top)**: تحسين سهولة التصفح.
- **معرض المشاريع (Portfolio Gallery)**: عرض احترافي لأهم القطاعات التي تخدمها الشركة.
- **زر الواتساب العائم**: تواصل فوري ومباشر مع خدمة العملاء.
- **كتالوج منتجات ذكي**: عرض مواصفات المنتجات مع ربط مباشر بالواتساب.
- **الوضع الليلي والنهاري**: تبديل سلس بين الأوضاع البصرية.
